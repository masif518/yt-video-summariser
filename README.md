# 🎥 YouTube Video Explainer AI

An AI-powered tool that **summarizes and explains any YouTube video** using **Google Gemini AI**, **YouTube Transcript API**, and a **Gradio web interface**.

Simply paste a YouTube link and the app will:

1. Extract the video ID
2. Fetch the transcript automatically
3. Send the transcript to Gemini AI
4. Generate a clear explanation of the video

---

# 🚀 Features

* 📺 Extracts transcript directly from YouTube videos
* 🤖 Uses **Google Gemini AI** for intelligent explanations
* 🧠 Converts long videos into easy-to-understand summaries
* 🌐 Simple **Gradio web interface**
* ⚡ Fast processing using **Gemini Flash model**

# 💡 How It Works

1. User enters a YouTube URL
2. The system extracts the **video ID**
3. Transcript is fetched using **YouTubeTranscriptAPI**
4. Transcript is sent to **Gemini AI**
5. Gemini generates a **clear explanation / summary**
6. Output is displayed in the **Gradio UI**
